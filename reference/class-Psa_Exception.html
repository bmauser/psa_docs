Class Psa_Exception

Psa_Exception class.

This class extends PHP Exception and adds Psa_Exception::log() method for writing PSA log. Other PSA exception classes extend this class. You can also extend this class for your custom exceptions. If logs are enabled in config.php file every exception message will be logged.
