Class Psa_Fatal_Error_Exception

Psa_Fatal_Error_Exception class.

If you raise Psa_Fatal_Error_Exception exception anywhere in your application, execution of the script will be terminated and message you provide as first argument to constructor will be logged.

Example:

class example_model extends Psa_Model {

    function example_method(){

        // some code here

        // die and write message to log
        if($fatal_error)
            throw new Psa_Fatal_Error_Exception('This message will be in logs');
        }
}
