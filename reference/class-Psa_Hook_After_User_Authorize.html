Class Psa_Hook_After_User_Authorize

Abstract class for Psa_Hook_After_User_Authorize hooks.

After a user is successfully authorized with Psa_User::authorize() method, psa_main() method of the hook object that extends this class will be called. You can write Psa_Hook_After_User_Authorize hooks by extending this class.

Example:

<?php
class my_hook extends Psa_Hook_After_User_Authorize{

    // you have to define psa_main() method in your child class
    function psa_main($user){

        // set some property to authorized user
        $user->favorite_background_color = 'white';

        $user->save();
    }
}
?>

When this example hook is registered every authorized user will have property called favorite_background_color with value 'white'.
