Class Psa_Hook_Before_User_Delete

Abstract class for Psa_Hook_Before_User_Delete hooks.

Before the user is deleted from the database with psa_delete_user() function, psa_main() method of the hook object that extends this class will be called. You can write Psa_Hook_Before_User_Delete hooks by extending this class.

Example:

<?php
class my_hook extends Psa_Hook_Before_User_Delete{

    // you have to define psa_main() method in your child class
    function psa_main($user){

        // this makes sense if $user is user ID
        $user = new Psa_User($user);
        $user->restore();

        // delete folder associated with the user
        rmdir ('/var/users/' . $user->username);
    }
}
?>

When this example hook is registered, before user deletion with psa_delete_user() function, folder associated with user will be deleted.

Note: argument passed to the psa_main() method can be an integer (user ID) or a string (username). It depends on what you pass to psa_delete_user() function.
