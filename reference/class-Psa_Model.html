Class Psa_Model

Models are the place for calculations, database interaction, data manipulation and other business logic.

Model classes contain methods that are called from the controller.

Model method should return a value which is passed to the view method in the controller. You can also pass results from the model to the view method with Psa_Result object because it is accessible from templates and view objects (see Psa_Dully_View or Psa_Smarty_View).

Example:

<?php
class my_model extends Psa_Model{

    $this->psa_database->query("SELECT * FROM my_table WHERE enabled = 1");
    return $this->psa_database->fetch_all();
}
?>


By extending Psa_Model class only Psa_Model::$psa_result, Psa_Model::$psa_registry and Psa_Model::$psa_database properties are set. So, if you do not need any of them in your model methods, your model class doesn't need to extend Psa_Model.
