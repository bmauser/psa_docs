Class Psa_Smarty_View

Class for views that use Smarty template engine.

Classes that extends Psa_Dully_View or Psa_Smarty_View class are place for for methods that will print results from your application. This can be HTML, XML or something else like plain text. View methods, like model methods, are called from controller. All application logic should be in model and in view you should put only part for displaying results (presentational logic) like template rendering code.

You can write Psa_Smarty_View views by extending this class. Here is an example:

<?php
class example_view extends Psa_Smarty_View{

    // Render even_numbers.tpl template and put it in "result" smarty value
    function even_numbers(){
        $rendered_result = $this->psa_smarty->fetch('even_numbers.tpl');
        $this->psa_smarty->assign("result", $rendered_result);
    }

    // Main layout render and display method.
    function generate_html(){

        $rendered_menu = $this->psa_smarty->fetch('menu.tpl');
        $rendered_header = $this->psa_smarty->fetch('header.tpl');

        $this->psa_smarty->assign('header', $rendered_header);
        $this->psa_smarty->assign('menu', $rendered_menu);

        echo $this->psa_smarty->fetch('example_app/main.tpl');
    }
}
?>
