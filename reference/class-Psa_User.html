Class Psa_User

User object class.

Here are some examples of using Psa_User object.


Usage examples:


1) Create a new user:

// user object
$user = new Psa_User('new');

// set username and password
$user->username = 'my_user';
$user->password = 'user_password';

// save user to the database
$user->save();


2) Authorize user by username and password:

$user = new Psa_User('my_user');
$user->authorize('user_password');


3) Extend Psa_User class. You'll probably want to extend Psa_User class and to add more columns to psa_user database table. In this example user_level, photo and email columns are added to psa_user table.

class MyUser extends Psa_User{

    public function __construct($user_id_or_username){
        parent::__construct($user_id_or_username, array('id', 'username', 'user_level', 'photo', 'email'));
    }
}

Code in some model method:

// create a new user
$user = new MyUser('new');
$user->username = 'my_user';
$user->user_level = 'admin';
$user->email = 'admin@host';
$user->save();


4) Change user's group membership:

// user object
$user = new MyUser(10); // MyUser extends Psa_User

// remove the user from all groups and put it in groups with ID 3,5 and 7
$user->remove_group('all');
$user->add_group(array(3,5,7));

Constructor method.

  • If called with 'new' as the first argument new user will be created when Psa_User::save() method is called.
  • If called with user ID as the first argument, Psa_User::$id value is set. In this case first argument must be an integer value.

# add_group( integer|array $group_id )

Puts user in a group or more groups if the $group_id argument is an array.

Puts user in a group or more groups if the $group_id argument is an array.

Group membership changes are immediately stored in the database and you don't have to call Psa_User::save() method after. If you creating a new user this method must be called after Psa_User::save() method. Psa_User::$id member variable must be set before calling this method. If you pass username to constructor, call Psa_User::restore() or Psa_User::authorize() method before to get the user ID from the database. Throws Psa_User_Exception on error.

Example:

$user = new Psa_User(123);
$user->add_group(5);

Parameters

$group_id
integer|array
ID or array with groups IDs.

Returns

integer
1 for success, -1 user already was in the group (or more groups) or group does not exists.
