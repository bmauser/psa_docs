Function prs

You can use this function instead of echo or print_r functions during development for debug output.

It will just return if PSA_CFG['develop_mode'] is FALSE. This function is wrapper for print_r function.

Example:

// print $array
prs($array);
Package: PSA
Author: Bojan Mauser <bmauser@gmail.com>
Located at lib/functions.php

Parameters summary

mixed $str
boolean $return_only = false When this parameter is set to TRUE, function will return the information rather than print it just like print_r() function.
