Function psa_autoload

This function is registered with the PHP spl_autoload_register() function as __autoload() implementation used to auto include .php files.

Thus, if you want to extend some class you don't have to include its file, it will be included automatically.

Note: Class autoloading will be working only in folders specified with $PSA_CFG['folders']['autoload'][] and $PSA_CFG['folders']['hook_autoload'][] configuration options. Also, file registration must be invoked to generate autoload_data.php file that contains array with all classes and their paths. There is a command line helper script register_files.php for that.

For example, you can just extend Psa_Model class without including its .php file:

<?php
class My_Model extends Psa_Model{
        function some_function(){
                // ... something here
        }
}
?>
