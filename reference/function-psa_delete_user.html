Function psa_delete_user

Deletes user from the database table.

Example:

// delete user with ID 123
psa_delete_user(123);

// delete user with usename 'my_user'
psa_delete_user('my_user');

// delete more users
psa_delete_user(array(1, 3, 5, 'my_user'))
Package: PSA
Author: Bojan Mauser <bmauser@gmail.com>
Located at lib/functions.php

Parameters summary

integer|array|string $user ID or username or array with user IDs or usernames. "all" to delete all users.

Return value summary

integer 0 for failure, 1 for success, -1 if an user (or more users) don't exist
