Psa_Db_Exception This exception will be raised by some methods from the Psa_PDO class.
Psa_Exception This class extends PHP Exception and adds Psa_Exception::log() method for logging. Other PSA exception classes extend this class. You can also extend this class for your custom exceptions. If logs are enabled in config.php file every exception message will be logged.
Psa_Fatal_Error_Exception If you raise Psa_Fatal_Error_Exception exception anywhere in your application, execution of the script will be terminated.
